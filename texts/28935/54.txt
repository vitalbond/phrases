<div><p>We are happy to announce the release of <strong>YUI 3.14.1</strong>! This release can be found on the <a href="http://yui.yahooapis.com/3.14.1/build/yui/yui-min.js" title="Yahoo CDN">Yahoo CDN</a>, through <a href="https://npmjs.org/package/yui" title="npm">npm</a>, and via <a href="http://yui.zenfs.com/releases/yui3/yui_3.14.1.zip">download</a>. We also have updated the <a href="http://yuilibrary.com/">YUI Library website</a> to reflect the changes in this release.</p>
<h2 id="whatsnew3141">What&rsquo;s New In This Release</h2>
<p>The primary purpose of this &ldquo;point&rdquo; release is to add IE11 and Android 4.4 to our <a href="http://yuilibrary.com/yui/environments/">Target Environments</a>. We&rsquo;ve been testing this release both manually and through CI to ensure that any new issues were addressed.</p>
<h3 id="ie11support3141">Support Added for IE11 and Android 4.4</h3>
<p>There were fixes to several tests this time around to resolve issues specific to IE11 and Android 4.4. This included an <a href="https://github.com/yui/yui3/pull/1482">issue</a> with <a href="http://yuilibrary.com/yui/docs/imageloader/">ImageLoader</a> fetching images that were out of date, an <a href="https://github.com/yui/yui3/pull/1501">issue</a> with skin loading, and <a href="https://github.com/yui/yui3/pull/1505">issues</a> with <a href="http://yuilibrary.com/yui/docs/editor/">Editor</a>.</p>
<p>Also, there were several code changes such as a <a href="https://github.com/yui/yui3/issues/1483">fix</a> by <a href="https://github.com/rgrove">Ryan Grove</a> to <a href="http://yuilibrary.com/yui/docs/api/classes/Plugin.ScrollInfo.html">ScrollInfo</a> and a <a href="https://github.com/yui/yui3/commit/680a9e80751528748eb64857eb80745eb66d0dd3">refactor</a> of the feature detection in <a href="http://yuilibrary.com/yui/docs/api/files/xml_js_xml-parse.js.html">XML-parse</a>.</p>
<h3 id="chartsregressionfix3141">Charts Regression Fix</h3>
<p>Also, there was an <a href="https://github.com/yui/yui3/issues/1475">issue</a> (introduced in <strong>YUI 3.14.0</strong>) in <a href="http://yuilibrary.com/yui/docs/charts/">Charts</a> that is fixed for this release. The issue was that the <code>_maxSize</code> property was not updated for a single series histogram and would cause the column/bar to disappear on mouseover.</p>
<h2 id="additionalinfo3141">Additional Information</h2>
<p>There was a total of 28 commits by 5 contributors for this release. You can find the details of the changes by reading the <a href="https://github.com/yui/yui3/wiki/YUI-3.14.1-Change-History-Rollup">Change History Rollup</a> as well as looking at the <a href="https://github.com/yui/yui3/compare/v3.14.0...v3.14.1">differences between 3.14.0 and 3.14.1</a> in GitHub. Over 9,000 tests in CI are run across our target environments for every commit to our source. If you run across an issue, please feel free to <a href="https://github.com/yui/yui3/issues/new">file it in GitHub</a> (after signing in) so we can continuously improve YUI!</p></div>
