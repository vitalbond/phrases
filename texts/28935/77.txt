<h2 id="release-news">Release News</h2>
<p><a href="https://github.com/yui/yeti/releases/tag/v0.2.27">Yeti 0.2.27 was released</a> this week. This release of Yeti brings code coverage reporting provided by <a href="https://github.com/gotwarlost/istanbul">Istanbul</a>. Check out the latest <a href="http://yeti.cx/">docs on Yeti&rsquo;s website</a>.</p>
<p><a href="https://github.com/yui/pure/releases/tag/v0.4.1">Pure 0.4.1 was also released</a> this week. This Pure release brings a bunch of little enhancements to your favorite mini CSS library.</p>
<h2 id="open-roundtable">YUI Open Roundtable</h2>
<p>This week&rsquo;s <a href="http://www.youtube.com/watch?v=4XhtP5arLHE">Open Roundtable</a> featured Drew Folta discussing Intl support in Modown. The talk was a great introduction to using Intl for localizing your Modown projects.</p>
