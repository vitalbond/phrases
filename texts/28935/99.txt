<div><p>We are announcing a new release candidate for you to try out: <strong>3.17.2-rc-1</strong>. You can find it on the <a href="http://yui.yahooapis.com/3.17.2-rc-1/build/yui/yui-min.js">Yahoo CDN</a>, via <a href="https://www.npmjs.org/package/yui">npm</a>, and download it via the <code>.zip</code> <a href="http://yui.zenfs.com/releases/yui3/yui_3.17.2-rc-1.zip">archive</a>. We have also updated the <a href="http://stage.yuilibrary.com/">staging website</a> for this release candidate.</p>
<p>This release candidate is targeted to fix <a href="http://yuilibrary.com/yui/docs/api/classes/Loader.html">Loader</a> regression issues from <strong>3.17.0</strong>. We found and fixed one such issue and released <strong>3.17.1</strong>. However other <a href="https://github.com/yui/yui3/pull/1837">issues</a> are being discovered, so please take this opportunity to download the release candidate and try it out in your own test environments, especially if you are having issues with <strong>3.17.1</strong>. Please send your feedback and <a href="https://github.com/yui/yui3/issues/new">file issues</a>. They have a direct positive impact on each and every release. You can check out what&rsquo;s changed in this release by viewing the <a href="https://github.com/yui/yui3/compare/v3.17.1...release-3.17.2-rc-1">code comparison</a> on GitHub.</p>
<p>When filing a bug for this release please include the <strong>module name</strong> and the <strong>YUI version</strong> in the title.</p></div>
