<div><p><em>Welcome to YUI Weekly, the weekly roundup of news and announcements from the YUI team and community. If you have any interesting demos or links you&rsquo;d like to share, feel free to leave a comment below.</em></p>
<ul><li>
<p>This week saw the release of <strong>YUI 3.12.0-beta-1</strong>, the first release candidate for the upcoming YUI 3.12 due out later this month. This release includes various improvements and fixes from 23 contributors across <a href="http://stage.yuilibrary.com/yui/docs/button/">Button</a>, <a href="http://stage.yuilibrary.com/yui/docs/charts/">Charts</a>, <a href="http://stage.yuilibrary.com/yui/docs/event/">Event</a>, <a href="http://stage.yuilibrary.com/yui/docs/model-list/">ModelList</a>, <a href="http://stage.yuilibrary.com/yui/docs/node/">Node</a>, <a href="http://stage.yuilibrary.com/yui/docs/tabview/">TabView</a>, <a href="http://stage.yuilibrary.com/yui/docs/template/">Template</a>, and <a href="http://stage.yuilibrary.com/yui/docs/tree/">Tree</a>. Check out the <a href="http://www.yuiblog.com/blog/2013/08/15/yui-3-12-0-release-candidate-1/">release announcement</a> for all the details. The <a href="https://github.com/yui/yui3/wiki/Development-Schedule#sprint-9-next-release">next milestone for Sprint 9</a> is the Aug 20th <a href="https://github.com/yui/yui3/wiki/Contribution-Standards#commit-freeze">commit freeze</a> pull request deadline.</p>
</li>
<li>
<p><a href="https://github.com/brunofarache/y3d/blob/master/README.md">y3d</a> is a <a href="http://en.wikipedia.org/wiki/WebGL">WebGL</a> module that is new to the <a href="http://yuilibrary.com/gallery/">Gallery</a> this week. Check out the <a href="http://brunofarache.github.io/y3d/">playground</a> for more demos. Nice work <a href="https://github.com/brunofarache">Bruno</a>!</p>
</li>
<li>
<p>At this week&rsquo;s <a href="https://github.com/yui/yui3/wiki/Open-Roundtable">Open Roundtable</a>, discussion topics included <a href="http://www.yuiblog.com/blog/2013/08/15/yui-3-12-0-release-candidate-1/">3.12-0-beta-1</a>, the <a href="https://groups.google.com/forum/#!msg/yui-contrib/BFMaCqS6oiE/rdvOZq-2ZVcJ">new release branch</a>, and a chat with <a href="http://github.com/rashadrussell">Rashad Russell</a> about his summer as a YUI intern. You can find a video of that Hangout <a href="https://www.youtube.com/watch?v=Hs_5OrjUoAk">here</a>.</p>
