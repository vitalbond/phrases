<ul><li>
<p>Now back from vacation, the team and community have been hard at work finishing up the next release of YUI. The release window is targeted to open up July 16th (next Tuesday) and barring any complications, will result in the release of <strong>YUI 3.11.0</strong>. To get a peak at what has changed, here&rsquo;s the <a href="https://github.com/yui/yui3/compare/v3.10.3...dev-3.x">v3.10.3&hellip;dev-3.x compare</a>. To summarize, it includes 551 commits from 19 authors and will include updates to <a href="https://yuilibrary.com/yui/docs/attribute/">Attribute</a>, <a href="https://yuilibrary.com/yui/docs/base">Base</a>, <a href="https://yuilibrary.com/yui/docs/calendar">Calendar</a>, <a href="https://yuilibrary.com/yui/docs/charts">Charts</a>, <a href="https://yuilibrary.com/yui/docs/datatable">Datatable</a>, <a href="https://stage.yuilibrary.com/yui/docs/paginator">Paginator</a>, and more, Thanks to all the contributors and testers who helped make this next release happen!</p>
</li>
<li>
<p>In the <em>&ldquo;Working with Pure CSS Modules&rdquo;</em> series (<a href="http://www.techrepublic.com/blog/webmaster/working-with-pure-css-modules-part-1/2743">part 1</a> and <a href="http://www.techrepublic.com/blog/webmaster/working-with-pure-css-modules-part-2/2751">part 2</a>), Ryan Boudreaux reviews the <a href="http://purecss.io/">Pure CSS library</a> and capabilities of each module.</p>
</li>

<p>At this week&rsquo;s <a href="https://github.com/yui/yui3/wiki/Open-Roundtable">Open Roundtable</a>, YUI summer interns Rashad (<a href="http://twitter.com/rashadrussell">@rashadrussell</a>) and Patrick (<a href="http://twitter.com/patjameson">@patjameson</a>) gave demonstrations of two projects they are working on, we discussed <a href="http://www.webrtc.org/">WebRTC</a>, Jeff demonstrated a new YUI example he&rsquo;s been working on, and we reacted to <a href="http://yuireactions.tumblr.com/">YUI Reactions</a>. Here&rsquo;s <a href="http://www.youtube.com/watch?v=2J0T5MT_G8c">the video</a> from that Hangout session.</p>
