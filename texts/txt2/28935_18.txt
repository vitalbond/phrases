<div><p><em>Welcome to YUI Weekly, the weekly roundup of news and announcements from the YUI team and community. If you have any interesting demos or links you&rsquo;d like to share, feel free to leave a comment below.</em></p>
<ul><li>
<p>After last week&rsquo;s batch of releases for <a href="http://yuilibrary.com/">YUI</a>, <a href="http://yeti.cx/">Yeti</a>, and <a href="http://purecss.io/">Pure</a>, the team has begun working towards the next cycle of releases. Announced this week was the <a href="https://github.com/yui/yui3/wiki/Development-Schedule#sprint-9-next-release">Sprint 9 development schedule</a>, which began on July 23rd and concludes with a stable release on August 27th. The next milestone for this sprint is the Aug 11th <a href="https://github.com/yui/yui3/wiki/Contribution-Standards#code-freeze">code freeze</a> pull request deadline. So for anyone working towards backwards incompatible updates, that is your deadline to submit a pull request.</p>
</li>
<li>
<p>Recently Github announced <a href="https://github.com/blog/1547-release-your-software">Releases</a>, a place for projects to host information and downloads for releases. You can now find all current and future release information for YUI at <a href="https://github.com/yui/yui3/releases">github.com/yui/yui3/releases</a> (<a href="https://github.com/yui/yui3/releases.atom">atom feed</a>).</p>
</li>
<li>
<p>At this week&rsquo;s <a href="https://github.com/yui/yui3/wiki/Open-Roundtable">Open Roundtable</a>, discussion topics included the recently released <a href="http://yuilibrary.com/yui/docs/paginator/">Paginator</a> component, a <a href="https://github.com/yui/yui3/pull/1029">Perf test refresh proposal</a>, a demo of <a href="http://twitter.com/rashadrussell">@rashadrussell</a>&lsquo;s Pure Layout Generator project, and a discussion on Github&rsquo;s <a href="https://help.github.com/articles/setting-the-default-branch">default branches</a>. You can find a video of that Hangout <a href="http://www.youtube.com/watch?v=NTA3fQYdOHE">here</a> or view it below.</p>
</li>
