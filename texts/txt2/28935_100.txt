<div><p>We are pleased to announce the release of <strong>YUI 3.17.2</strong>. You can find this release on the <a href="http://yui.yahooapis.com/3.17.2/build/yui/yui-min.js">Yahoo CDN</a>, through <a href="https://npmjs.org/package/yui">npm</a>, and via a <a href="http://yui.zenfs.com/releases/yui3/yui_3.17.2.zip">downloadable <code>.zip</code> archive</a>. We have also updated the <a href="http://yuilibrary.com/">YUI Library website</a> to reflect the changes in this release.</p>
<h2 id="whatsnew3172">What&rsquo;s New In This Release</h2>
<p>This point release fixes an <a href="https://github.com/yui/yui3/pull/1837">issue with Loader</a> that was discovered after the release of <strong>3.17.1</strong>. If you are currently using <strong>3.17.1</strong> and have encountered Loader issues you are encouraged to update to this version. If you are using a version earlier than <strong>3.17.1</strong> (for example <strong>3.16.0</strong>) and you want to find out about what&rsquo;s new since then, check out our <a href="http://www.yuiblog.com/blog/2014/05/21/yui-3-17-1-released/">blog post for <strong>3.17.1</strong></a>.</p>
<h2 id="additionalinformation3171">Additional Information</h2>
<p>There were a total of <a href="https://github.com/yui/yui3/compare/v3.17.1...v3.17.2"><strong>12</strong> commits by <strong>3</strong></a> contributors for this release. You can find details about the changes in this release by checking out the <a href="https://github.com/yui/yui3/wiki/YUI-3.17.2-Change-History-Rollup">Change History Rollup</a> for <strong>3.17.2</strong>. For every commit made to our source, there are over 9,000 tests run against our <a href="http://yuilibrary.com/yui/environments/">target environments</a>. If you discover an issue, please <a href="https://github.com/yui/yui3/issues/new">file it in GitHub</a> (you&rsquo;ll need to sign in first). If you would like to contribute tests, documentation, code fixes, or new features, please check out our <a href="https://github.com/yui/yui3/wiki/Contributing-to-YUI">Contributing to YUI Wiki</a> entry.</p></div>
