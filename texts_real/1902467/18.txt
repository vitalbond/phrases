<div><p>The <a href="http://www.w3.org/2012/ldp/">Linked Data Platform (LDP) Working Group</a> invites implementation of the Candidate Recommendation of <a href="http://www.w3.org/TR/2014/CR-ldp-20140619/">Linked Data Platform 1.0</a>. This document describes a set of best practices and simple approach for a read-write Linked Data architecture, based on HTTP access to web resources that describe their state using the RDF data model. Learn more about the <a href="http://www.w3.org/2013/data/">Data Activity</a>.</p></div>
