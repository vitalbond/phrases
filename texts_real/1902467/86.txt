<div><p>The <a href="http://www.w3.org/2009/dap/">Device APIs Working Group</a> and <a href="http://www.w3.org/2011/04/webrtc/">Web Real-Time Communications Working Group</a> have published a First Public Working Draft of <a href="http://www.w3.org/TR/2014/WD-selection-api-20141007/">Selection API</a>.  This specification extends the Media Capture and Streams specification to allow a depth stream to be requested from the web platform using APIs familiar to web authors. Learn more about the <a href="http://www.w3.org/2007/uwa/">Ubiquitous Web Applications </a>.</p></div>
