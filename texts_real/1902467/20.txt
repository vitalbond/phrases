<div><p>The <a href="http://www.w3.org/2010/webperf/">Web Performance Working Group</a> has published a Last Call Working Draft of <a href="http://www.w3.org/TR/2014/WD-beacon-20140624/">Beacon</a>. This specification defines an interoperable means for site developers to asynchronously transfer small HTTP data from the User Agent to a web server. Comments are welcome through <strong>29 July 2014</strong>.</p>
<p>The <a href="http://www.w3.org/2010/webperf/">Web Performance Working Group</a> also updated the Candidate Recommendation of <a href="http://www.w3.org/TR/2014/CR-resource-timing-20140624/">Resource Timing</a>. This specification defines an interface for web applications to access the complete timing information for resources in a document.</p>
<p>Learn more about the <a href="http://www.w3.org/2006/rwc/">Rich Web Client Activity</a>.</p></div>
