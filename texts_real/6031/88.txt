<div><p>Today the <a href="http://www.w3.org/XML/Query/">XQuery Working Group</a> published a Last Call Working Draft of <a href="http://www.w3.org/TR/2014/WD-xquery-31-20141007/">XQuery 3.1</a> and <a href="http://www.w3.org/TR/2014/WD-xqueryx-31-20141007/">XQueryX 3.1</a>. Additional supporting documents were published jointly with the <a href="http://www.w3.org/Style/XSL/">XSLT Working Group</a>: a <a href="http://www.w3.org/TR/2014/WD-xpath-31-20141007/">Last Call Working Draft of XPath 3.1</a>, together with <a href="http://www.w3.org/TR/2014/WD-xpath-functions-31-20141007/">XPath Functions and Operators</a>, <a href="http://www.w3.org/TR/2014/WD-xpath-datamodel-31-20141007/">XQuery and XPath Data Model</a>, and <a href="http://www.w3.org/TR/2014/WD-xslt-xquery-serialization-31-20141007/">XSLT and XQuery Serialization</a>. XQuery 3.1 and XPath 3.1 introduce improved support for working with JSON data with map and array data structures as well as loading and serializing JSON; additional support for HTML class attributes, HTTP dates, scientific notation, cross-scaling between XSLT and XQuery and more. Comments are welcome through <strong>7 November 2014</strong>. Learn more about the <a href="http://www.w3.org/XML/Activity">XML Activity</a>.</p></div>
