<div><p>The <a href="http://www.w3.org/International/core/">Internationalization Working Group</a> has published a Working Draft of <a href="http://www.w3.org/TR/2014/WD-predefined-counter-styles-20140821/">Predefined Counter Styles</a>. This document describes numbering systems used by various cultures around the world and can be used as a reference for those wishing to create user-defined counter styles for CSS. Learn more about the <a href="http://www.w3.org/International/">Internationalization Activity</a>.</p></div>
