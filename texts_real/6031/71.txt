<div><p>The <a href="http://www.w3.org/2011/webappsec/">Web Application Security Working Group</a> has published a Working Draft of <a href="http://www.w3.org/TR/2014/WD-mixed-content-20140916/">Mixed Content</a>. This specification describes how user agents should handle rendering and execution of content loaded over unencrypted or unauthenticated connections in the context of an encrypted and authenticated document. Learn more about the <a href="http://www.w3.org/Security/">Security Activity</a>.</p></div>
