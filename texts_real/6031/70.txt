<div><p>The <a href="http://www.w3.org/2012/ldp/">Linked Data Platform (LDP) Working Group</a> has published a Last Call Working Draft of <a href="http://www.w3.org/TR/2014/WD-ldp-20140916/">Linked Data Platform 1.0</a>. This document describes a set of best practices and simple approach for a read-write Linked Data architecture, based on HTTP access to web resources that describe their state using the RDF data model. Comments are welcome through 7 October.</p>
<p>The group also published a Group Note of <a href="http://www.w3.org/TR/2014/NOTE-ldp-acr-20140916/">LDP Access Control</a>. This note discusses use cases and requirements for Access Control for the Linked Data Platform Working Group. It also outlines a charter for developing a standard for HTTP-based access control.</p>
