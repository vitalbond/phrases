<div><p>WordPress 3.7 Beta 2 is now available for download and testing.&nbsp;This is software still in development, so we&nbsp;don&rsquo;t recommend that you run it on a production site.</p>
<p>This has been a quiet beta period. We&rsquo;re hoping to get some more testers for <strong>automatic background updates</strong>, which will occur for security and minor releases (like updating from 3.7 to 3.7.1). It&rsquo;s really easy to test this, as Beta 2 will update* each day to the latest development version and then email you the results. If something goes wrong, you can report it &mdash; it&rsquo;s that simple. To get the beta, try the&nbsp;<a href="http://wordpress.org/extend/plugins/wordpress-beta-tester/">WordPress Beta Tester</a>&nbsp;plugin (you&rsquo;ll want &ldquo;bleeding edge nightlies&rdquo;). Or you can&nbsp;<a href="http://wordpress.org/wordpress-3.7-beta2.zip">download the beta here</a>&nbsp;(zip). Check out <strong>Dashboard &rarr; Updates</strong> to see if your install is eligible for background updates. WordPress won&rsquo;t update if, for example, you&rsquo;re using version control like SVN or Git.</p>
<p>For more of what&rsquo;s new in version 3.7,&nbsp;<a title="WordPress 3.7 Beta 1" href="http://wordpress.org/news/2013/09/wordpress-3-7-beta-1/">check out the Beta 1 blog post</a>. In Beta 2, we further increased the stability of background updates and also added about 50 bug fixes, including a fix for Internet Explorer 11 in the visual editor.</p>
<p>If you think you&rsquo;ve found a bug, you can post to the&nbsp;<a href="http://wordpress.org/support/forum/alphabeta">Alpha/Beta area</a>&nbsp;in the support forums. Or, if you&rsquo;re comfortable writing a bug report,&nbsp;<a href="http://core.trac.wordpress.org/">file one on the WordPress Trac</a>. There, you can also find&nbsp;<a href="http://core.trac.wordpress.org/report/5">a list of known bugs</a>&nbsp;and&nbsp;<a href="http://core.trac.wordpress.org/query?status=closed&amp;group=component&amp;milestone=3.7">everything we&rsquo;ve fixed</a>.</p>
<p>Happy testing!</p>
