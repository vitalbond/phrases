<div><p>The second release candidate of WordPress 3.7 is now available for testing!</p>
<p>Those of you already testing WordPress 3.7 will be updated automatically to RC2. (<em>Nice.</em>) If you&rsquo;d like to start testing, there&rsquo;s no time like the present! Try the&nbsp;<a href="http://wordpress.org/extend/plugins/wordpress-beta-tester/">WordPress Beta Tester</a>&nbsp;plugin (you&rsquo;ll want &ldquo;bleeding edge nightlies&rdquo;) or&nbsp;<a href="http://wordpress.org/wordpress-3.7-RC2.zip">download the release candidate here</a>&nbsp;(zip).&nbsp;Please post to the&nbsp;<a href="http://wordpress.org/support/forum/alphabeta/">Alpha/Beta area in the support forums</a>&nbsp;if you think you&rsquo;ve found a bug, and if any known issues are raised, you&rsquo;ll be able to&nbsp;<a href="http://core.trac.wordpress.org/report/5">find them here</a>.</p>
<p>Developers,&nbsp;please test your plugins and themes against WordPress 3.7. If there is a compatibility issue, let us know as soon as possible so we can deal with it before the final release.</p>
<p>For more on WordPress 3.7, check out the&nbsp;<a href="http://wordpress.org/news/2013/10/wordpress-3-7-release-candidate/">announcement post for Release Candidate 1</a>.</p></div>
