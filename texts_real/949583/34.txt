<div><p>The <a href="http://www.w3.org/2011/webappsec/">Web Application Security Working Group</a> has published a First Public Working Draft of <a href="http://www.w3.org/TR/2014/WD-mixed-content-20140722/">Mixed Content</a>. This specification details how user agents can mitigate risks to security and privacy by limiting a resource???s ability to inadvertently communicate in the clear, or to expose non-public resources to the web at large. This specification describes how and why user agents disallow rendering and execution of content loaded over unencrypted or unauthenticated connections in the context of an encrypted and authenticated document. Learn more about the <a href="http://www.w3.org/Security/">Security Activity</a>.</p></div>
