<div><p>The <a href="http://www.w3.org/html/wg/">HTML Working Group</a> invites implementation of the Candidate Recommendation of <a href="http://www.w3.org/TR/2014/CR-2dcontext-20140821/">HTML Canvas 2D Context</a>. This specification defines the 2D Context for the HTML canvas element. The 2D Context provides objects, methods, and properties to draw and manipulate graphics on a canvas drawing surface. Learn more about the <a href="http://www.w3.org/MarkUp/Activity">HTML Activity</a>.</p></div>
