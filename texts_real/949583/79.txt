<div><p>The User Agent Accessibility Guidelines Working Group (<a href="http://www.w3.org/WAI/UA/">UAWG</a>) today published updated Working Drafts of <a href="http://www.w3.org/TR/2014/WD-UAAG20-20140925/">User Agent Accessibility Guidelines (UAAG) 2.0</a> and <a href="http://www.w3.org/TR/2014/WD-UAAG20-Reference-20140925/">UAAG 2.0 Reference: Explanations, Examples, and Resources for User Agent Accessibility Guidelines 2.0</a>. UAAG defines how browsers, media players, and other “user agents” should support accessibility for people with disabilities and work with assistive technologies. It is introduced in the <a href="http://www.w3.org/WAI/intro/uaag">UAAG Overview</a>. Read about the <a href="http://www.w3.org/WAI/">Web Accessibility Initiative (WAI)</a>.</p></div>
