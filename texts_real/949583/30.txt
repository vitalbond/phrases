<div><p>The <a href="http://www.w3.org/International/core/">Internationalization Working Group</a> has published a Working Draft of <a href="http://www.w3.org/TR/2014/WD-charmod-norm-20140715/">Character Model for the World Wide Web: String Matching and Searching</a>. This document builds upon on Character Model for the World Wide Web 1.0: Fundamentals to provide authors of specifications, software developers, and content developers a common reference on string identity matching on the World Wide Web and thereby increase interoperability. Learn more about the <a href="http://www.w3.org/International/">Internationalization Activity</a>.</p></div>
