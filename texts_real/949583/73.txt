<div><p>The <a href="http://www.w3.org/2012/ldp/">Linked Data Platform (LDP) Working Group</a> has published a First Public Working Draft of <a href="http://www.w3.org/TR/2014/WD-ldpatch-20140918/">Linked Data Patch Format</a>. Linked Data Patch Format (LD Patch) defines a language for expressing a sequence of operations to apply to Linked Data resources; it is suitable for use with the HTTP PATCH, a method to perform partial modifications to Web resources. Learn more about the <a href="http://www.w3.org/2013/data/">Data Activity</a>.</p></div>
