<div><p>The CSS Working Group and the SVG Working Group have published a Last Call Working Draft of <a href="http://www.w3.org/TR/2014/WD-geometry-1-20140626/">Geometry Interfaces Module Level 1</a>. This specification provides basic geometric interfaces to represent points, rectangles, quadrilaterals and transformation matrices that can be used by other modules or specifications. CSS is a language for describing the rendering of structured documents (such as HTML and XML) on screen, on paper, in speech, etc. Comments are welcome through <strong>7 August 2014</strong>. Learn more about the <a href="http://www.w3.org/Style/">Style Activity</a> and the <a href="http://www.w3.org/Graphics/">Graphics Activity</a>.</p></div>
