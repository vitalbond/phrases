<div><p><a href="http://www.w3.org/TR/WCAG-EM/">Website Accessibility Conformance Evaluation Methodology (WCAG-EM) 1.0</a> was published today as a W3C Note by the Web Content Accessibility Guidelines Working Group (<a href="http://www.w3.org/WAI/GL/">WCAG WG</a>) and Evaluation and Repair Tools Working Group (<a href="http://www.w3.org/WAI/ER/">ERT WG</a>), through the joint <abbr title="Web Content Accessibility Guidelines">WCAG</abbr> 2.0 Evaluation Methodology Task Force (<a href="http://www.w3.org/WAI/ER/2011/eval/eval-tf">Eval TF</a>). WCAG-EM describes an approach for evaluating how websites, including Web applications and websites for mobile devices, conform to <a href="http://www.w3.org/WAI/intro/wcag">WCAG 2.0</a>. WCAG-EM provides guidance for individuals and organizations that evaluate web accessibility, and is also useful for those that have others evaluate their website. Learn more from the <a href="http://www.w3.org/WAI/eval/conformance">WCAG-EM Overview</a> and about the <a href="http://www.w3.org/WAI/">Web Accessibility Initiative (WAI)</a>.</p></div>
